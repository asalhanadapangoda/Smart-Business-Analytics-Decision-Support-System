# SBADSS - Smart Business Analytics & Decision Support System

SBADSS is a modern, full-stack enterprise platform designed for SMEs to track business performance, analyze trends, and receive AI-powered decision support.

## 🏗️ Architecture
- **Backend**: Spring Boot 3 (Java 17/21) + PostgreSQL + Redis
- **AI Microservice**: FastAPI (Python 3.9+) + Prophet + scikit-learn
- **Frontend**: React + Vite + TailwindCSS v4 + Zustand

---

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have the following installed:
- [Java 17+](https://adoptium.net/)
- [Node.js 20+](https://nodejs.org/)
- [Python 3.9+](https://www.python.org/)
- [PostgreSQL](https://www.postgresql.org/) (Running on port 5432)
- [Redis](https://redis.io/) (Running on port 6379)

### 2. Database Setup
1. Create a database named `sbadss` in PostgreSQL.
2. The tables will be automatically generated by Hibernate when you start the backend.

### 3. Run the AI Microservice (Python)
The AI service handles forecasting and NLP chatbot logic.
```bash
cd ai-service
# Create a virtual environment (optional but recommended)
python -m venv venv
source venv/bin/activate # Windows: .\venv\Scripts\activate
# Install dependencies
pip install -r requirements.txt
# Start the service
uvicorn app.main:app --reload --port 8000
```

### 4. Run the Backend (Spring Boot)
Ensure the `dev` profile is active (default).
```bash
cd backend
# If you have Maven installed:
mvn spring-boot:run
# Or use your IDE (IntelliJ/Eclipse) to run SbadssBackendApplication.java
```
- API will be available at: `http://localhost:8080/api/v1`
- Swagger UI (if enabled): `http://localhost:8080/swagger-ui.html`

### 5. Run the Frontend (React)
```bash
cd frontend
# Install dependencies
npm install
# Start the development server
npm run dev
```
- The dashboard will be available at: `http://localhost:5173`
